# Portfolio Website

Personal portfolio for Gracious James Eluvathingal. The site is designed to present public proof of work in AI automation, backend engineering, cloud systems, and infrastructure-oriented projects without turning into a template-heavy personal site.

## Stack

- Astro
- Tailwind CSS v4
- Static project case-study pages

## What the site covers

- Featured case studies drawn from public and sanitized professional work
- Experience summary with measurable outcomes
- Public proof points such as the TARS podcast clip
- Personal operating dashboard for work, learning, life, and systems metrics
- Dangling local dashboard assistant that explains the metrics without requiring an API key
- A recruiter-friendly entry point into current focus areas and technical direction

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:4321`.

## Build

```bash
npm run build
```

## Notes

- Most recent production work is represented as sanitized case studies rather than source code because the underlying systems are private.
- Resume assets and project media live in `public/`.
